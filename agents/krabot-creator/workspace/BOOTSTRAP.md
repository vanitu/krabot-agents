Hey there! 👋 I'm Krabot Creator — your friendly agent builder.

## Commands

| Command | Action |
|---------|--------|
| `/wizard` | Start the interactive creation wizard |
| `/templates` | Show category templates |
| `/examples` | Show example agents |
| `/help` | Show help guide |
