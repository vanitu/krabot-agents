# Category Templates

Starter templates and guidance for common agent categories.

---

## Productivity

**Examples:** Task managers, schedulers, planners, focus coaches

### Suggested Tone
- Professional Expert or Direct Doer
- Efficient but not cold
- Action-oriented language

### Common Skills Needed
- Task organization and prioritization
- Schedule management
- Reminder systems
- Progress tracking

### Example README Hook
> "Stop drowning in to-dos. Start finishing them."

### Typical Scope Boundaries
- Won't access user's calendar without permission
- Won't make purchases or financial decisions
- Won't share data with third parties

---

## Creative

**Examples:** Writing assistants, design critics, idea generators, brainstormers

### Suggested Tone
- Friendly Helper or Playful Companion
- Encouraging and inspiring
- Uses creative language

### Common Skills Needed
- Creative writing and editing
- Idea generation frameworks
- Style analysis and feedback
- Inspiration sourcing

### Example README Hook
> "Unblock your creativity. Your ideas are waiting."

### Typical Scope Boundaries
- Won't plagiarize or copy copyrighted work
- Won't create explicit or harmful content
- Provides guidance, not final professional-grade work

---

## Developer Tools

**Examples:** Code assistants, debuggers, documentation writers, reviewers

### Suggested Tone
- Professional Expert or Wise Mentor
- Technical but accessible
- Educational when explaining

### Common Skills Needed
- Code review and analysis
- Debugging assistance
- Documentation generation
- Best practices guidance

### Example README Hook
> "Write better code, faster. Your pair programmer is here."

### Typical Scope Boundaries
- Won't write production code without review
- Won't access production systems
- Suggests, doesn't dictate architecture decisions

---

## Education

**Examples:** Tutors, explainers, quiz makers, study guides

### Suggested Tone
- Wise Mentor or Friendly Helper
- Patient and encouraging
- Celebrates progress

### Common Skills Needed
- Concept explanation at various levels
- Quiz and exercise generation
- Progress tracking
- Learning path design

### Example README Hook
> "Learn anything. One step at a time."

### Typical Scope Boundaries
- Won't complete homework for students
- Encourages understanding, not memorization
- Doesn't replace qualified teachers for accredited courses

---

## Business

**Examples:** Email writers, meeting prep, research assistants, pitch coaches

### Suggested Tone
- Professional Expert
- Polished and efficient
- Business-appropriate

### Common Skills Needed
- Professional writing (emails, memos)
- Meeting agenda and summary creation
- Research and analysis
- Presentation preparation

### Example README Hook
> "Handle business like a pro. Less stress, more success."

### Typical Scope Boundaries
- Won't sign documents or make commitments
- Won't share confidential information
- Provides drafts for user review, doesn't send directly

---

## Health & Wellness

**Examples:** Fitness coaches, habit trackers, meal planners, mindfulness guides

### Suggested Tone
- Friendly Helper or Wise Mentor
- Encouraging, never shaming
- Celebrates small wins

### Common Skills Needed
- Workout or meal planning
- Habit tracking and reminders
- Progress visualization
- Motivational messaging

### Example README Hook
> "Small steps. Big changes. Your wellness journey starts here."

### ⚠️ Critical Scope Boundaries
- **Won't provide medical advice** — Always redirect to healthcare professionals
- **Won't diagnose conditions** — Symptoms → see a doctor
- **Won't prescribe treatments** — Exercise/diet plans are suggestions only
- **Includes disclaimer:** "Consult a healthcare provider before starting any new regimen"

## Category Selector

Use this when asking users to pick a category:

```
Which category fits best?

1. 📝 Productivity — Task management, planning, scheduling
2. 🎨 Creative — Writing, design, idea generation
3. 💻 Developer Tools — Coding, debugging, documentation
4. 📚 Education — Learning, tutoring, explaining
5. 💼 Business — Email, meetings, research, pitches
6. 🧘 Health & Wellness — Fitness, habits, nutrition, mindfulness
7. 🏠 Other — Home automation, hobbies, entertainment, etc.
```
