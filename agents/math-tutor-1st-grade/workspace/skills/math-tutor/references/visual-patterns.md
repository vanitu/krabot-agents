# Визуальные схемы для объяснения

## Сложение

### Прибавить 1:
```
🍎 + 🍎 = 🍎🍎
1 + 1 = 2

🚗 + 🚗🚗 = 🚗🚗🚗
1 + 2 = 3
```

### Прибавить 2:
```
🌟🌟 + 🌟🌟 = 🌟🌟🌟🌟
2 + 2 = 4

🦋🦋🦋 + 🌟🌟 = 🦋🦋🦋🌟🌟

```

### Сложение через числовую линию:
```
2 + 3 = ?

Стартуем тут 👇
1️⃣2️⃣➡️3️⃣➡️4️⃣➡️5️⃣

Прыгнули 3 раза ➡️➡️➡️

```

## Вычитание

### Отнять 1:
```
🍎🍎🍎 - 🍎 = 🍎🍎
3 - 1 = 2

🚗🚗🚗🚗 - 🚗 = 🚗🚗🚗

```

### Отнять 2:
```
🌟🌟🌟🌟🌟 - 🌟🌟 = 🌟🌟🌟
5 - 2 = 3

🦋🦋🦋🦋 - 🌟🌟 = 🦋🦋

```

### Вычитание через числовую линию:
```

1️⃣2️⃣3️⃣4️⃣5️⃣

```

